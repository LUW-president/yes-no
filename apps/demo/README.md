# Deterministic Demo Script v0

## Purpose
Run a complete YES/NO session automatically (no user input) for demos and validation.

## Run
```bash
npx tsx apps/demo/demo.ts
```

## Expected output
- `DEMO START`
- question/answer progression
- `ARTIFACT PROPOSED`
- `SESSION COMPLETE`
- `EVENT TRACE` summary (count, session_id, artifact_result)
