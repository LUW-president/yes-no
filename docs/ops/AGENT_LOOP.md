# AGENT_LOOP.md

## Execution Loop
1. Read `AGENT_DIRECTIVE.md`
2. Read `PROJECT_STATE.md`
3. Identify current phase
4. Execute the highest-priority approved task
5. Run tests
6. Commit changes
7. Report using required reporting format

## Operating Rules
- do not rely on chat memory
- repo docs are the source of truth
- pause on ambiguity when action is restricted
