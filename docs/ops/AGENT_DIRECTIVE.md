# AGENT_DIRECTIVE.md

## Project
YES/NO by The Navy Blue Hub

## Mission
Build the YES/NO platform: a radically simple AI system where users only answer YES or NO to questions.

## Non-Negotiable Product Principles
- One question at a time
- Binary interaction only (YES / NO)
- No prompt interface
- Simplicity over features
- The questions are the product
- Silence and timing are part of the experience

## System Baseline Summary
- Internal v0 baseline complete
- Core deterministic stack implemented: protocol, question engine, memory engine, session orchestrator
- Operational surfaces implemented: CLI runner, UX harness, packaging, demo, status dashboard
- Integration surfaces implemented: reference bridge, mobile/web adapters, mobile/web reference apps
- Full repository baseline test suites currently passing

- create code
