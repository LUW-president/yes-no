# One-Glance Session Checklist

Use once per participant.

- [ ] Participant type selected (first_time / repeat)
- [ ] Device selected (mobile / desktop)
- [ ] Decision topic entered
- [ ] Session started without moderator intervention
- [ ] Session completed to final summary
- [ ] Time to complete captured (seconds)
- [ ] Gate result captured (GO / REVIEW / NO_GO)
- [ ] Interview Q1 captured (start clear yes/no)
- [ ] Interview Q2 captured (summary understandable yes/no)
- [ ] Interview Q3 captured (friction note)
- [ ] JSONL line appended
