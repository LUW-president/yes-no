# First User Guide

YES/NO is a single-session decision app.

## What the app does
- Helps you clarify one decision at a time
- Uses short YES/NO prompts
- Returns a deterministic decision summary (GO / REVIEW / NO_GO)

## How to start a session
1. Open the app URL
2. (Optional) enter a decision topic
3. Click **Start Session**
4. Answer prompts with **Yes** or **No**
5. Review final summary

## Example decisions
- Should I move to another city?
- Should I accept this offer?
- Should I postpone this purchase?

## Expected session flow
- Start session
- 2–4 binary prompts
- Final summary with:
  - gate result
  - guard status
  - confidence
  - primary reason

## Scope note
Current public mode is focused on **single-session clarity, speed, and reliability**.
Continuous assistant behavior is intentionally out of scope for this phase.
