# Discovery Question Set v1

Purpose: deterministic binary funnel to narrow immediate user need.

## Question Funnel (MVP)
- Q1: Is there something you want right now?
- Q2: Can it happen right now?
- Q3: Does it involve another person?
- Q4: Is it something physical?
- Q5: Would it change how you feel?
- Q6: Is it something you usually enjoy?
- Q7: Is it something practical?
- Q8: Is it urgent?
- Q9: Does it involve your environment?
- Q10: Is it something you usually outsource?

## Notes
- Binary only (YES/NO).
- Questions are deterministic and sequence-based in this preparation phase.
- Runtime wiring is intentionally out of scope for this document.
