# Product Deck Outline v0 — YES/NO

## Product Thesis
YES/NO is a radically simple AI interface where the system asks questions and the user answers only YES or NO.

The product removes prompts, menus, and complexity, replacing them with a guided sequence of binary decisions.

Key concept:

**"The questions are the product."**

## Problem
Current AI interfaces create avoidable friction:
- prompt writing is difficult
- users face blank-page syndrome
- complex tools overwhelm users
- interaction with AI often feels mechanical

YES/NO solves this by replacing prompting with guided decision flow.

## Interaction Model
YES/NO interaction system:
- one question at a time
- binary responses only
- black-screen minimal UI
- gesture input (circle / cross)
- calm pacing and silence as part of UX

## Why Now
The moment is right:
- AI systems are powerful but interfaces remain primitive
- mobile interaction patterns are evolving
- users increasingly prefer simple experiences
- gesture and minimal UI paradigms are emerging

YES/NO proposes a new interface layer for AI.
