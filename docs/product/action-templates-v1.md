# Action Templates v1

Purpose: define deterministic proposal templates linked to intent buckets.

## Templates
- Would you like me to order {favorite_food}?
- Would you like me to play {favorite_music}?
- Would you like the latest {favorite_podcast} episode?
- Would you like me to book {service_provider}?
- Would you like me to contact {doctor_or_care_team}?
- Would you like me to hire {service_type}?
- Would you like me to turn off the lights?
- Would you like me to find a babysitter?
- Would you like me to call 911?

## Notes
- Templates use personalization variables.
- This document defines contract surface only (no runtime wiring).
