(circle / cross, yes / no)

## Layer 2 — Decision engine
(trace, confidence, guard, gate)

## Layer 3 — Decision memory
(history of your choices)

## Layer 4 — Personal pattern model
(what your decisions reveal)

(proposes actions)

## Notes
