# Known Limitations (v0)

Current baseline limitations:

- console/terminal-only UI
- no real mobile native UI yet
- no real browser UI yet
- no gesture recognition yet
- no notification system yet
- no production persistence yet
- no hosted deployment yet

Additional practical constraints:
- reference app bridge has minimal transport/security hardening
- reference interfaces are validation harnesses, not production UX
- artifact lifecycle controls are limited in current v0 interaction surface
