# Status Dashboard v0

## Purpose
Provide a minimal internal status view of the YES/NO baseline, components, commands, test state, and next milestone.

## Run
```bash
npx tsx ops/dashboard/statusDashboard.ts
# or
npx tsx apps/cli/yesno.ts status
```

## What it shows
- baseline release marker
- stable component list
- supported command list
- test status summary
- next milestone focus
